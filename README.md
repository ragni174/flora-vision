# 🌿 FloraVision

A premium cinematic plant-inspired frontend experience built using **React**, **Tailwind CSS**, and **Framer Motion**, inspired by luxury ecommerce interfaces and modern editorial web design aesthetics.

FloraVision is designed to create an immersive and visually rich user experience through:
- cinematic dark visuals
- atmospheric lighting
- glassmorphism effects
- organic layouts
- smooth animations
- premium typography
- modern responsive design

The project focuses heavily on frontend polish, interaction design, and creating a high-end aesthetic inspired by modern digital product experiences.

- 🌱 Luxury botanical-inspired UI

# ⚙️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/ragni174/flora-vision.git
```

---

## Navigate to the Project

```bash
cd flora-vision
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

The application will run locally on:

```bash
http://localhost:5173
```

---

# 📦 Production Build

```bash
npm run build
```
